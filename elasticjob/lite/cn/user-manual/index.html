ElasticJob > 用户手册

本章导航

  • ElasticJob 提供了简单易用的运维平台，方便用户操作及查询作业。

  • 配置手册介绍了如何通过 Java 和 Spring 两种方式配置。

  • 一般情况 ElasticJob 是通过平均分配算法的分片策略数据的，但也可以选择哈希及轮转等策略，或者自己定义作业分片策略。

  • 为了便于记录、查询、统计及监控作业运行情况，ElasticJob 提供了事件追踪功能，也可自行对作业运行状态监控。

  • 由于无法在生产环境调试，通过dump可以把作业运行相关信息导出，方便开发者调试分析。

  • 最后，ElasticJob 还提供了其它扩展功能，如：作业监听器、自诊断修复、定制化处理及操作手册等。
